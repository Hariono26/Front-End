Redux step by step:

- yg bisa diimport dari 'react-redux' => { Provider, connect }
- yg bisa diimport dari 'redux' => { combineReducers, createStore }

1. siapkan reducer (jgn lupa index.js)
	a. pada reducernya
		- buat initial state
		- buat function reducer dg berbagai case yg diinginkan
	b. pada index.js reducer
		- import combineReducers
		- import semua reducer
		- combine semua reducer

2. siapkan action (jgn lupa index.js)
	a. pada actionnya
		- buat berbagai action yang diinginkan(jgn lupa type dan payload kalau ada data yang mau dikirim)
	b. pada index.js action
		- export semuanay (*) dari action

3. siapkan redux di index.js yang utama
	- import createStore, import Provider, import reducers yg sudah dicombine
	- buat variabel untuk menampung global state pakai createStore
	- subscribe variabel global state untuk mengconsole log global state setiap react dipanggil (optional)
	- di dalam reactDOM.render, bungkus App dengan Provider
	- di dalam Provider, masukkan props store berisi global state

4. redux siap disajikan
	- import connect di page/component dimana kamu mau menggunakan data dari redux
	- import action yang kamu mau gunakan di page/component
	- buat function mapStateToProps, kemudian masukkan ke connect beserta action yg ingin dipakai
