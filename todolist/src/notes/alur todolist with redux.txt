1. App.js dipanggil => melakukan render component :
	a. navigationbar
	b. todopages
2. navigationbar => merender componentnya dan diconnect ke redux
3. todopages => merender componentnya dan diconnect ke redux
4. componentDidMount jalan, lalu menjalankan fetchData
5. fetchData mengambil data dari json-server, lalu mengirimkan data tersebut melalui action getData
6. action getData akan menghasilkan sebuah object yang berisi 2 properti:
	a. type : (disesuaikan dengan case yang ada di reducer)
	b. payload : (berisi data dari hasil axios)
7. object hasil dari action getData akan diteruskan ke todoReducer
8. sesampainya di todoreducer. case yg dijalankan adalah yg sesuai dengan property type yang diterima (pada kasus ini adalah type: GET_DATA)
9. case yg sesuai(GET_DATA) mengisi data activities dengan data hasil get dari axios yang dibawa melalui property payload
10. ketika data di todo reducer berubah maka render akan terpanggil lagi
11. sehingga data yang ditampilkan sekarang adalah hasil dari data redux yang sudah berubah
